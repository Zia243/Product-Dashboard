# Product Dashboard Application

A modern product dashboard built with React 19, TypeScript, Tailwind CSS, Zustand for state management, and Recharts for data visualization. The application interacts with DummyJSON API for authentication and product data.

- **Authentication**: Secure login/logout

- **Product Management**: Browse, search, and view product details
- **Pagination**: Efficient product listing with pagination support
- **User Profile**: View and manage user information
- **Modern UI**: Clean, responsive design with Tailwind CSS
- **Type Safety**: Full TypeScript support
